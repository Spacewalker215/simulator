# Autonomous Driving Simulator

See [the documentation](https://ut-av.pages.dev/) for [setting up the simulator](https://ut-av.pages.dev/tools/simulation/) and [reinforcement learning](https://ut-av.pages.dev/tutorials/reinforcement_learning/).

## Usage

To use the simulator, setup the `~/roboracer_ws`:

```bash
cd ~
git clone https://github.com/ut-av/roboracer_ws.git
cd roboracer_ws
./scripts/checkout.sh
```

## Development

Install [UnityHub](https://unity.com/download), clone this project into `~/roboracer_ws/simulator`, and open it in UnityHub.
